# राजस्थान भूगोल — मेगा क्वेस्ट 🗺️

RSSB **Basic Computer Instructor (Paper-I)** के लिए राजस्थान भूगोल का एक इंटरैक्टिव लर्निंग ऐप — एक ही HTML फ़ाइल, पूरी तरह ऑफलाइन, बिना किसी फ्रेमवर्क या इंटरनेट कनेक्शन के चलता है।

**Live:** https://jayant-1211.github.io/Rajasthan-MEGA-Quest/

## क्या-क्या है

| भाग | विवरण |
|---|---|
| 📖 अध्याय | 11 अध्याय — सामान्य परिचय, भौतिक स्वरूप, जलवायु, नदियाँ, झीलें, सिंचाई-बाँध, जल संरक्षण, मिट्टी, खनिज, वनस्पति, मैप मास्टरी |
| ⚡ फ्लैश कार्ड | 142 कार्ड — रिकॉल, रिवर्स रिकॉल, रिक्त स्थान, तुलना, कालक्रम, कथन-आधारित, ट्रैप कार्ड |
| ⚔️ बैटल मोड | 140 MCQ — RSSB पैटर्न, हर सवाल में व्याख्या + ट्रैप नोट |
| 🗺️ मैप लैब | 41 जिलों का इंटरैक्टिव नक्शा, लेयर सिस्टम, स्मार्ट सर्च, मैप क्विज़ |
| 🐉 फाइनल बॉस | सभी अध्यायों से मिक्स 40 सवाल, 3 जान |
| 🦁 XP सिस्टम | लेवल, बैज, कॉम्बो — प्रगति ब्राउज़र में सेव रहती है |

हर अध्याय एक ही शिक्षण-क्रम पर चलता है: सरल भाषा में व्याख्या → क्यों ज़रूरी है → वास्तविक उदाहरण → परीक्षा पॉइंट → RSSB ट्रैप → मेमोरी ट्रिक → 30 सेकंड रिवीजन।

## चलाने का तरीका

ऊपर दिया गया लिंक खोलो — बस इतना ही। या `index.html` डाउनलोड करके किसी भी ब्राउज़र में खोल लो; इंटरनेट के बिना भी चलेगा।

## तकनीकी

एक ही फ़ाइल — inline CSS + vanilla JavaScript + inline SVG मानचित्र। कोई बिल्ड स्टेप नहीं, कोई डिपेंडेंसी नहीं, कोई external API नहीं। प्रगति `localStorage` में सेव होती है।

## नोट

अध्ययन सामग्री (कोचिंग नोट्स व PDF) इस रिपॉज़िटरी में शामिल नहीं है — केवल ऐप प्रकाशित किया गया है।
