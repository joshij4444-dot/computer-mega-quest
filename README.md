# कंप्यूटर मेगा क्वेस्ट 💻

RSSB **Basic Computer Instructor — Paper-II** के लिए एक इंटरैक्टिव लर्निंग ऐप। पूरी तरह ऑफलाइन, बिना किसी फ्रेमवर्क, बिल्ड स्टेप या इंटरनेट कनेक्शन के चलता है।

## क्या-क्या है

| भाग | विवरण |
|---|---|
| 📚 यूनिट | हर यूनिट में — 📖 पढ़ो (लेसन) • ⚡ फ्लैश कार्ड • ⚔️ बैटल (MCQ) |
| ⚡ फ्लैश कार्ड | 10 प्रकार — रिकॉल, रिवर्स रिकॉल, रिक्त स्थान, कथन-आधारित, मैच, कालक्रम, तुलना, Code Output, True/False, ट्रैप कार्ड |
| ⚔️ बैटल मोड | RSSB पैटर्न MCQ — हर सवाल में व्याख्या + ट्रैप नोट + कठिनाई + टॉपिक टैग + PYQ समानता |
| 🧪 इंटरैक्टिव लैब | SQL Playground (आगे — Excel, MS Word Ribbon, DSA विज़ुअलाइज़र, OSI मॉडल, आदि) |
| 🐉 फाइनल बॉस | सभी यूनिट से मिक्स 40 सवाल, 3 जान |
| 🏅 XP सिस्टम | लेवल, बैज, कॉम्बो — प्रगति ब्राउज़र में सेव रहती है |

हर यूनिट एक ही शिक्षण-क्रम पर चलती है: सरल भाषा में व्याख्या → गहराई से → यह concept बना ही क्यों → वास्तविक मिसाल → डायग्राम → तुलना तालिका → मेमोरी ट्रिक → RSSB ट्रैप → PYQ इनसाइट → 30 सेकंड रिवीज़न।

## चलाने का तरीका

`index.html` को किसी भी ब्राउज़र में खोल लो — बस इतना ही। इंटरनेट के बिना भी चलेगा।

⚠️ `index.html` अकेला काफ़ी नहीं — साथ में `css/`, `js/` और `data/` फ़ोल्डर भी रखने होंगे।

## तकनीकी

कोई बिल्ड स्टेप नहीं, कोई डिपेंडेंसी नहीं, कोई external API नहीं। Vanilla JS (classic scripts — इसीलिए `file://` से भी सीधा चलता है) + inline SVG डायग्राम। प्रगति `localStorage` में सेव होती है (key: `cmquest`)।

```
index.html            शेल — मार्कअप + लोड ऑर्डर
css/style.css         डिज़ाइन सिस्टम
data/content.js       LESSON / CARDS / QB कंटेनर
data/paper2/u-*.js    हर यूनिट (लेसन + कार्ड + MCQ)
js/state.js           XP, लेवल, बैज, localStorage, नेविगेशन
js/chapters.js        यूनिट रजिस्ट्री + ग्रिड
js/sheet.js           यूनिट शीट (3 टैब)
js/battle.js          MCQ बैटल इंजन
js/labs/*.js          इंटरैक्टिव लैब
js/app.js             boot
archive/              पुराना Paper-I कंटेंट (सुरक्षित)
```

## आर्काइव

पुराना **राजस्थान भूगोल (Paper-I)** कंटेंट **हटाया नहीं गया** — `archive/paper1_geography/` में पूरा सुरक्षित है: लेसन, फ्लैश कार्ड, MCQ, मैप लैब, 41-ज़िला एटलस, मैप क्विज़, मैप CSS और स्टैंडअलोन `Rajasthan_MEGA_Quest.html` (जो अकेला ही चल जाता है)। ज़रूरत पड़ने पर वापस लाया जा सकता है।
